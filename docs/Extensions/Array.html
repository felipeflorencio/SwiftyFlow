public extension Array
  • subscript(safe:)

    Very helpful helper in order to get a item from array in safe way that you don’t need to first check if indice is there and access latter, just set the indice and will return if array have.

    Usage Example:

        let exampleArray = ["item1", "item2", "item3", "item4", "item5", "item6"]
    let returnIfExist = exampleArray[safe: 3]

    If not there will return nil, so you can use with 'guard' to

    Declaration

    Swift

    subscript(safe index: Int) -> Element? { get }

    Return Value

    Element of the index that you are looking for if exist at that indice.
